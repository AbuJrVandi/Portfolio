Online Banking Application

A secure and robust banking system built with Python and OOP principles.

Project Overview

This project is a comprehensive online banking application developed using Python and Object-Oriented Programming (OOP). It provides a secure, intuitive platform for users to manage their finances, perform transactions, and view their account history.

The system is designed with a focus on security, data integrity, and user experience, simulating a real-world banking environment through a clean command-line interface.

Key Features

  • Secure User Authentication: Robust login system to ensure secure access to user accounts.
  • Account Management: Users can view account details, check balances, and manage personal information.
  • Fund Transfers: Seamlessly transfer funds between accounts with transaction validation.
  • Transaction History: Access a detailed history of all transactions, including deposits, withdrawals, and transfers.
